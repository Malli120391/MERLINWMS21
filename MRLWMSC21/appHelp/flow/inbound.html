    1. Inward Order
      1. To create an Inward Order; Account, Tenant and Supplier must be created as mentioned earlier in masters module.
      2. Account and Tenant must be mapped to the Supplier.
      3. Now go to Orders Inward Order, the below page is displayed.
      4. Click on “Add+” button and the below popup is displayed.
      5. Order type, Inward Date, Tenant and Supplier are mandatory. Once the data is given and saved Inward Order Ref. No. is auto generated.
      6. An alert message is displayed as shown below.
      7. The status changes to “OPEN”.
      8. Now in above screen click on “Add material +” button and a screen is displayed as shown below.
      9. The part # in material details must be created initially in Item master. (Refer Masters manual 2.5)
      10. Once the mandatory fields i.e. part #, UoM Qty. and Quantity are given click on “Update” button.
      11. An alert message is displayed as shown below.
      12. In the above screen click on “Add Invoice +” button in Supplier Invoice details and the below screen is displayed.
      13. Invoice Number and Invoice date are mandatory. Once the supplier invoice details are given click on “Update” button and an alert message is displayed as shown below.
      14. Now in the same screen in Material Details, Invoice column click on Add Invoice which is in red (indicating invoice to be added).
      15. The below popup is displayed.
      16. Click on “Add Invoice Details” button and the below screen is displayed.
      17. Now give the mandatory fields and submit them.
      18. The submitted invoice details are displayed in the table as shown below.
      19. Now click on “Update” button beside the material details table.
      20. An alert message is displayed as shown below.
      21. Now in the same screen in Material Details, Invoice column the “Add Invoice” turns blue (indicating invoice is added and updated)
    2. New Inbound
      1. In the main menu go to Inbound New Inbound and the below screen is displayed.
      2. Give the mandatory fields (Warehouse must be initially created in the Account (Refer Master manual 2.1)) and click on initiate shipment.
      3. The status changes to “Initiated” as shown below.
      4. To update the date and time, in the same page give the date and time and click on “Update” button.
      5. Now click on “Add Order / Invoice +” in the Inward Order/Invoice Details panel.
      6. Below displayed screen is shown.
      7. The Order Number must be the same as Inward Order Ref # in Orders>Inward Order>Inward Order Header Details.
      8. Once the Inward Order/Invoice Details are given, click on “Update” button.
      9. An alert message is displayed as shown below.
      10. Now go to 3. Shipment Expected Details and click on “Update” and the below shown alert message is displayed.
      11. Now if any docks are updated in Master Data>Account>Warehouse, those docks can be released here.
      12. If no docks are updated there and if clicked on Dock in receiving dock Management panel then an alert message is displayed as shown below.
      13. Now coming to 6.Shipment Received Details, the below screen is displayed.
      14. The below shown fields are displayed. Select the receiving store and click on “Receive” button.
      15. An alert message is displayed as shown below.
      16. Now RTR must be done. In 6.Shipment Received Details click on RTR[1] and the below screen is shown.
      17. Now click on “Receive” which navigates the page to Goods In as shown below.
      18. Now select the PO Number, in Putaway list panel give all the mandatory fields as shown below.
      19. Now click on “Receive” button and an alert message is displayed as shown below.
      20. Each Item must be received in the same manner.
      21. Now go to Inbound Search Inbound, the below data is displayed.
      22. Search for the PO Number and click on modify icon.
      23. The below screen is displayed and the status will be shown as “Shipment Received”.
      24. Now scroll down to 8.GRN Details and click on “Add GRN +” button.
      25. The below screen is displayed.
      26. Give all the mandatory fields and click on “Line Items” button. A popup is displayed as shown below.
      27. Now check the Action checkbox and click on “Save” button.
      28. Then click the “Update” button and no the status changed to “GRN Updated” as shown below.
      29. As the complete quantity is received no discrepancy is shown here. But if partial quantity is received then the discrepancy panel is displayed as shown below.
      30. Here the discrepancy must be checked and saved.
      31. Now in 10.Shipment Verification panel shown below, click on “verify” button to close the shipment.
      32. An alert message is displayed as shown below.
      33. With this Inbound process is completed.
