    1. Outward Order
      1. To start an Outbound process, check if current stock is available in the inventory or not. (To check the current stock go to Main menu Inventory Current stock and search with the available filters)
      2. Account and Tenant must be mapped to the Supplier.
      3. Click on “Add +” button.
      4. The below screen is displayed.
      5. Give the outward order header details..
      6. Click on “Save” button.
      7. The below screen is displayed.
      8. Once the outward header details are saved, Outward Ref No. is displayed.
      9. Now the status is shown as “OPEN”.
      10. Now in the same page, scroll down and go to Customer PO details.
      11. The below screen is displayed.
      12. Now click on “Add Customer PO +” button.
      13. Give the mandatory fields and submit.
      14. The below screen is displayed with an alert message.
      15. Now click on “Add Material” button in the Material Details panel.
      16. Give the mandatory fields and submit.
      17. The below screen is displayed with an alert message.
    2. Outbound
      1. Now in the main menu go to Outbound>Create Outbound.
      2. The below screen is displayed.
      3. Now select an SO and click on “Get SO’s” button.
      4. The below screen is displayed.
      5. Now check the SO or SO’s for which the outbound is to be created and select warehouse.
      6. Click on “Create Outbound” button and the below screen is displayed with an alert message.
      7. Now go to Outbound Release Outbound. Here the OBD’s that are to be released are shown with a blinking dot near the OBD date as shown below.
      8. Here OBD’s can be searched using the Tenant and OBD Number filters.
      9. Now search for the related OBD in the displayed list and click on the release icon.
      10. The below screen is displayed with the options of carrying out the OBD process or VLPD (Group OBD) process.
      11. The delivery quantity can be edited here.
      12. Now check the selected OBD and click on:
        1. Release for OBD
          1. A pop-up is displayed as shown below.
          2. Fill the data in the popup and click on “Release OBD” button.
          3. The below screen is displayed with an alert message.
          4. Now go to Outbound Search Outbound and the below screen is displayed.
          5. Now click on edit icon in the Modify column and the below screen is displayed.
          6. Now the status is shown as “On Hold”.
          7. Scroll down to 3.Pick n Check details and click on “Pick note” near the top right corner of the screen.
          8. The below shown delivery pick note is displayed.
          9. Scroll the grid to the right side and click on the pick note icon.
          10. The below screen is displayed.
          11. Enter quantity and click on “Pick” button.
          12. The picked quantity is shown in the table below in the same popup as shown below.
          13. The picking can be cancelled here by clicking the delete icon in the above shown popup.

            14. For release VLPD the following process follows:

        2. Release for VLPD
          1. The below shown screen is displayed with an alert message.
          2. Now go to Outbound Group Outbound.
          3. Check, give the mandatory fields and click on “Create Group OBD”.
          4. The below screen is displayed.
          5. Now go to group outbound list, and the below screen is displayed.
          6. Click on “Initiate Pick Up” button and the below screen is displayed.
          7. Click on the pick icon shown in the above page.
          8. The below screen is displayed.
          9. Now click on the pick icon shown in the above page.
          10. The below screen is displayed.
          11. Now a popup is displayed as shown below.
          12. Give the mandatory fields and click on “Pick” button and the below screen is displayed showing the updated picked quantity and pending quantity.
          13. Now once the page is refreshed, verify icon will be displayed in the grid as shown below.
          14. Click on verify icon and the below shown alert message is displayed.
      13. Now go to Outbound Search Outbound, the below screen is displayed.
      14. Search for the SO number and click on modify icon. The below screen is displayed.
      15. Now give the mandatory fields and click on “Update and send to PGI” button.
      16. The below screen is displayed with an alert message.
      17. Now go to 4.PG details, give the mandatory fields and update.
      18. The below screen is displayed with an alert message.
      19. Now go to 6.Packing Details, give the mandatory fields and update.
      20. The below screen is displayed with an alert message.
      21. Now go to 7.Delivery Details, give the mandatory fields and update.
      22. The below screen is displayed with an alert message.
      23. Now the user can check the status of the outbound in 1. Initiate Outbound Delivery and status will be shown as “Delivered”. The outbound process closes here.
