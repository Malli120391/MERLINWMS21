    1. Material Transfer
      1. To create an Inward Order; Account, Tenant, Materials must be created as mentioned earlier in masters module.
      2. Material must be mapped to the Tenant.
      3. Now go to Orders House Keeping Material Transfer, the below page is displayed.
      4. User can perform four types of Material Transfers:
        1. Bin To Bin
        2. SL to SL
        3. Expiry Material Movement
        4. Fast Moving Loc Fulfilling
      5. Bin To Bin
        1. User can transfer materials from one bin to another bin by using this material transfer type.
        2. Click on “Add+” button and the below popup is displayed.
        3. Tenant and Transfer Type are mandatory.
        4. Here Transfer type should be Bin To Bin.
        5. Once the data is given and saved, Ref. No. is auto generated.
        6. An alert message is displayed as shown below.
        7. Here Ref. No. should be unique.
        8. Fill Transfer Details info. Here, Part#, Location, Quantity and To Location are mandatory fields.
        9. After filling details, user needs to click on Save Details button as shown below:
        10. Now click on Back To List button.
        11. Here select Transfer Type – Bin To Bin and Status – Initiated and click on Search button.
        12. Now for respective record, click on Initiate button. Below screen is displayed.
        13. Now select Transfer Type – Bin To Bin and Status as ‘In Process’. Below screen is displayed:
        14. Click on Edit icon for the same record. Below screen is displayed:
        15. Here select ‘To Container’ from auto-complete.
        16. Click on Transfer button. Below alert should display and transfer of material qty should happen from one bin to other bin.
        17. Now user can check in Current stock whether material qty is deducted from one bin and added to another bin as shown below:
      6. SL to SL
        1. User can transfer materials from one SLoc to another SLoc by using this material transfer type.
        2. Click on “Add+” button and the below popup is displayed.
        3. Tenant and Transfer Type are mandatory.
        4. Here Transfer type should be SL To SL.
        5. Once the data is given and saved, Ref. No. is auto generated.
        6. An alert message is displayed as shown below.
        7. Here Ref. No. should be unique.
        8. Check the Current stock page:
        9. Fill Transfer Details info. Here, Part#, Location, Quantity, From SL and To SL are mandatory fields.
        10. After filling details, user needs to click on Save Details button as shown below:
        11. Now click on Back To List button.
        12. Here select Transfer Type – SL To SL and Status – Initiated and click on Search button.
        13. Now for respective record, click on Initiate button. Below screen is displayed.
        14. After Initiating, check the Current stock report.
        15. Transfer is happened to given To SLoc. User can check Current Stock report to verify the transfer.
      7. Expiry Material Movement
        1. Auto transfer order is generated every midnight for expired material.
          2. Process Explanation with example:-
        2. Suppose today (17-Aug-2018) user has performed Inbound for material MI having expiry date ’20-Aug2018’.
        3. It means on 20-August-2018 M1 material will be expired.
        4. 20 Aug midnight, one job will run on backend and transfer order will get generated for material M1.
        5. For checking that generated transfer order, user needs to select Transfer Type – Expiry Material Movement and status – Initiated and click on Search button as shown below:
        6. Click on Initiate button. Below screen will display
        7. Now select Transfer Type – Expiry Material Movement and Status – In Process and click on Search button.
        8. Now Transfer orders for Expired Material should display which are in process.
        9. Click on Edit link for that record.
        10. Transfer Detail record should come with all the details of material and quarantine location
        11. User needs to select To Container and click the the transfer button.
      8. Fast Moving locations Fulfillment
        1. Process Explanation with example
          1. Go to Orders Inward Orders
          2. Create PO: PO1800138 and INB: FKW1800028 for 300 qty of ‘RawMaterial’ and receive at location Z103A0101(which is not a fast moving location)
          3. Create one more PO: PO1800139 and INB: FKW1800029 for 300 qty of RawMaterial’ and receive at location Z202A0203(FastMoving Location).
          4. Go to Orders Outward Orders
          5. Create SO: SO1800103 with 290 qty
          6. Created and release OBD(1800104).
          7. While picking, system should suggest FastMoving Loc i.e Z202A0203 for picking item RawMaterial.
          8. Once user starts picking items and on fast moving location stock goes below min threshold for the bin.
          9. Auto transfer order should get created to fullfil bin stock to its max threshold for the fast moving bin.
          10. Now user needs to Select Transfer Order: FastMoving Loc Fulfilling and Status – Initiated, user will get auto generated transfer order for FastMoving bin material.
