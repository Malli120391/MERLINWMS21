    1. Customer Returns
      1. To initiate the Customer Returns go to Menu Orders Returns Customer Returns.
      2. The below screen is displayed:
      3. Give the mandatory fields (Tenant, OBD Number, Invoice and Store) and click on “Get Details”.
      4. The below screen is displayed with the fetched data:
      5. Now check the required part number and click on “Return” button.
      6. Now the Customer return order is made successfully and an Inbound is created.
      7. The Inbound Ref No. is displayed in the alert message after clicking on “Return” button as shown in the below screen.
      8. Now Inbound can be done following the Inbound User Manual.
    2. Supplier Returns

      3. Give the mandatory fields (Tenant, OBD Number, Invoice and Store) and click on “Get Details”.

      5. Now check the required part number and click on “Return” button.
