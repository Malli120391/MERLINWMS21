    • Why Job Order is not getting created ?
      • If LCM of MOP’s of child items is greater than the available qty of Parent item in current stock then user will not be able to create Job Order.
    • Why Job Order is not getting created even when the LCM of MOP’s of child items is greater than the available qty of Parent item in current stock ?
      • Available stock for the child items should be less than the min thresholds’s for respective items.
    • Which job order creation method should be preferred if user needs to fulfill stock as per the threshold values given to items ?
      • Auto job order creation method should be preferred if user needs to fulfill stock as per the threshold values given to items
    • Which job order creation method should be preferred if user needs stock for particular items ?
      • Manual Job Order creation method should be preferred if user needs stock for particular items.
    • Where can I check threshold for items ?
      • Go to Masters Item Master launch Search respective item click on edit link of respective item navigate to Replenishment grid having header(De-Nesting Stock Replenishment), there user can check min and max threshold for respective items.
    • Where can I check available stock for items ?
      • Go to Inventory Current Stock launch Search respective item. User will get the available qty for respective item.
    • Where to check the LCM of MOP’s ?
      • Go to Job List BOM Master launch Click on edit link for respective BOM. Check under ‘BOM Header Details’ section. User will get all the details regarding child Item MOP’s.
    • How to create an auto joborder ?
      • It is a backend process, every midnight, one SP will run at backend and it will create job orders for all the items below than their configured min thresholds. User can check for auto created job order by navigating through Job List BOM Master launch Job list
    • How to add a stock for a particular file ?
      • If there is requirement for a particular item, user can created manual job order. Navigate to Job List Job List launch Click on Add New + Fill the mandatory details on form and click Save.
